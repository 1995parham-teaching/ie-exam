\قسمت{از \lr{URL} به تقاضا}

فرض کنید مرورگر شما می‌خواهد به URL زیر تقاضا ارسال کند:

\begin{latin}
\begin{minted}[]{output}
  https://httpbin.org/get?hello=world#H
\end{minted}
\end{latin}

برای اینکار از تقاضای زیر استفاده می‌کند. مواردی که با <> مشخص شده‌اند را مقداردهی کنید.

\begin{latin}
\begin{minted}[]{http}
GET <path> HTTP/1.1
Host: <host>
\end{minted}
\end{latin}
