\قسمت{رویدادها}

سپهر یک برنامه نویس فرانت است، از او در یک مصاحبه خواسته شده است
که تابع \متن‌لاتین{setBgColor}
را که تعریف آن در ادامه آمده است با استفاده از \متن‌سیاه{جاوا اسکریپت}
و \متن‌سیاه{بدون}
تغییر در کد \متن‌لاتین{HTML}
به عنوان یک هندلر روی یک دکمه ثبت کند. آیا این امر ممکن است؟
دقت داشته باشید که این دکمه حتما این تابع را با پارامتر \متن‌لاتین{\متن‌ایتالیک{orange}}
فراخوانی می‌کند.
شما مجاز به نوشتن هر میزان کد جاوا اسکریپت که صلاح می‌دانید هستید.

\begin{latin}
\begin{minted}[]{js}
function setBgColor(color) {
  document.getElementById('change-my-color')
                    .style.backgroundColor = color;
}
\end{minted}
\end{latin}
