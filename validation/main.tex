\section{صحت‌سنجی}

\paragraph{}
پروژه پیاده‌سازی یک وب‌سایت برای سازمان ثبت احوال به شما واگذار شده است.
توضیح دهید که استفاده از صحت‌سنجی\LTRfootnote{validation} در سمت کلاینت چه مزایایی برای شما دارد و آیا تنها همین صحت‌سنجی کفایت کرده و شما نیاز به پیاده‌سازی در سمت سرور ندارید؟

\paragraph{}
۳ نمره

\answer{
  \begin{itemize}
          \item اشاره به کاهش بار سرور
          \item اشاره به عدم کفایت صحت‌سنجی سمت کلاینت
          \item اشاره به تجربه کاربری بهتر
  \end{itemize}
}
